## 🎯 Objetivo

Ferramenta de estudo **ativa e eficaz** para preparação de avaliações. Totalmente offline, sem dependências externas, funciona direto no navegador.

---

## 📚 Conteúdo Coberto

O quiz aborda **80 questões** distribuídas nos seguintes tópicos:

| Tema | Questões | %  |
|------|----------|-----|
| **Regra de Negócio** | 15-20 | 18-25% |
| **Requisito Funcional** | 40-50 | 22-25% |
| **Requisito Não Funcional** | 50-60 | 25-30% |
| **Diagrama de Casos de Uso** | 40-50 | 21-25% |
| **Diagrama de Classes** | 50-60 | 25-30% |

### Tópicos Específicos

✅ Classificação de requisitos (RF vs RNF)  
✅ Tipos de RNF (Desempenho, Confiabilidade, Segurança, Usabilidade, Portabilidade, etc)  
✅ Relacionamentos UML (Include, Extend, Generalização)  
✅ Elementos de Casos de Uso (Atores, Fronteira do Sistema, Fluxos)  
✅ Estrutura de Classes (Atributos, Métodos, Visibilidade)  
✅ Multiplicidades e Cardinalidades  
✅ Agregação vs Composição  
✅ Herança e Polimorfismo  

---

## ✨ Características

🎲 **Questões Aleatórias** — A ordem muda a cada sessão  
🔀 **Opções Embaralhadas** — Cada questão tem opções em ordem diferente  
📊 **Estatísticas Detalhadas** — Acompanhe seu progresso por tema  
🎯 **Filtros Inteligentes** — Escolha estudar por tema ou modo (novo, revisão)  
💾 **Persistência de Dados** — Progresso salvo automaticamente no navegador  
🌙 **Dark Theme** — Interface elegante otimizada para longas sessões  
📱 **Responsivo** — Funciona em mobile, tablet e desktop  
🚀 **Offline-First** — Nenhuma conexão necessária
